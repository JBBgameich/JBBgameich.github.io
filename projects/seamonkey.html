Seamonkey aus Sparky-Quellen installieren (für Debian testing)

  • melden sie sich in einem Terminal (xterm, konsole, lxterminal, qterminal etc.) als root an.
  • erstellen sie die Datei:
    nano /etc/apt/sources.list.d/sparky.list
  • Fügen sie

    ## sparky

    deb http://sparkylinux.org/repo testing main

    in die Datei ein.
  • Updaten sie die Paketeliste
    apt-get update
  • Jetzt müssen sie noch den GPG-Schlüssel hinzufügen:
    apt-get install sparky-keyring
  • Als leztes natürlich noch das eigendliche Installieren:
    apt-get install seamonkey
    . Alternativ können sie natürlich auch grafische Paketemanger wie Synaptics oder apper verwenden.

    Installation von Addons:

    Methode 1

  • Starten Sie Ihre Kopie von SeaMonkey

  • Klicken Sie auf dieser Seite auf den Link "Zu Seamonkey hinzufügen".
  • Sie bekommen eventuell eine Informationsmeldung am oberen Rand unserer Webseite angezeigt, die Ihnen mitteilt, dass die Installation von Software durch www.jbbgameich.github.io verhindert wurde. Klicken Sie die Schaltfläche zum Installieren der Software, um fortzusetzen.

    • Methode 2

  • Alternativ klicken sie auf "Download", und wählen sie in Ihrem Dateimanger (Dolhpin, Thunar, Windows Explorrer) nach einem Rechtsklick "Seamonkey".

  • SeaMonkey installiert nach Bestätigung der Informationsmeldung das deutsche Paket.
  • Wählen Sie im Mozilla-Browser den Menüpunkt "Edit" ("Bearbeiten"), darunter "Preferences..." ("Einstellungen..."). Dort öffnen Sie im der Kategorie "Appearance" ("Erscheinungsbild").
  • Dort wählen Sie die Sprache "German" ("Deutsch") aus dem Auswahlfeld im unteren Bereich des Fensters.
  • Klicken Sie OK, beenden Sie SeaMonkey und starten Sie das Programm neu. Tun Sie dies, so wird die Oberfläche in deutscher Sprache angezeigt.
